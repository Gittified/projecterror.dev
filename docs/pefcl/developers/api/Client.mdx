## Exports

PEFCL exposes a few useful exports that be used from other resources to integrate better with the financing system.
The exports support both callback & promises.

## Events

### `pefcl:newTransaction`

```typescript
interface Account {
  id: number;
  number: string;
  balance: number;
  isDefault: boolean;
  accountName: string;
  ownerIdentifier: string;
  role: 'owner' | 'admin' | 'contributor';
  type: 'personal' | 'shared';
  createdAt?: string;
}

interface Transaction {
  id: number;
  toAccount?: Account;
  fromAccount?: Account;

  amount: number;
  message: string;
  type: 'Outgoing' | 'Incoming';

  updatedAt?: string;
  createdAt?: string;
}
```

<Tabs>

AddEventHandler('pefcl:newTransaction', function(transaction)
